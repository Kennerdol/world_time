🌍 World Time
A simple Flutter application that displays the current local time for cities and countries around the world. Select a location to instantly view its current time based on its time zone.

✨ Features
🌎 Browse locations worldwide
🕒 Display accurate local time
🔍 Search for cities and countries
📱 Clean and responsive Flutter UI
⚡ Fast and easy to use
🚀 Getting Started
Prerequisites
Flutter SDK
Dart SDK
Installation
git clone https://github.com/Kennerdol/world_time.git
cd world_time
flutter pub get
flutter run
